## 本地开发

### 环境

-   [Node.js](https://nodejs.org/en/) Version >= 10
-   [npm](https://www.npmjs.com/) Version >= 6

### 安装

```
pnpm install
```

### 开发

```
# 当前目录 ./packages/<%= projectName %>/
npm run dev

# 或者项目根目录 ./
pnpm run --filter  @univerjs/<%= projectName %> dev
```

### 打包

```
# 当前目录 ./packages/<%= projectName %>/
npm run build

# 或者根目录 ./
pnpm run --filter  @univerjs/<%= projectName %> build
```
